# riak_kv

This is a fork of [riak_kv](https://github.com/basho/riak_kv) altered to have the functionality of EunomiaKV Plus. 

## Dependencies

- Erlang R16B02
- GNU-style build system

## Compilation 

```
# Install dependencies and compile project
$ make compile 

# Clean build files 
$ make clean
```
